Para ejecutar cada uno de los codigos, debe de abrir el CMD(Terminal) y buscar
la ruta de la carpeta del proyecto en especifico, en este caso:

C:\Users\......\ADA06_ImpOrdenamiento

Luego de estar en la carpeta del programa, primero debe compilar las clases,
para eso, debe escribir en la terminal:

javac *.java

Escribiendo esto en la terminal estando dentro de las carpeta,
generara los .class del proyecto, Ya por ultimo, para ejecutarlo, lo unico
que debe hacer es escribir en la terminal:

java App MarioKart2.csv

Esto hara que, que el App lea el Archivo Csv "MarioKart2", en caso de querer
usar otro CSV, simplemente cambie el nombre "MarioKart2" por el nombre de su
archivo en el comando. 
Ejemplo: Quiero ponerle mi Archivo CSV llamado "Visitas":

java App Visitas.csv
